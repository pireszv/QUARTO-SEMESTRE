A utilização do CSS in-line facilita na visualização do estilo juntamente ao campo, tornando mais simples os ajustes rápidos. Além disso, permite que fique tudo em um só arquivo, o que otimiza o tempo. Entretanto, não é interessante para projetos grandes, uma vez que pode demonstrar um baixo desempenho comparado ao CSS externo e se tornar mais complexo se houver muitos itens que terão a mesma estilização. Também deixa o HTML menos intuitivo.

Já com a utilização do CSS externo, apesar de parecer mais complicado para páginas muito pequenas/protótipos e requerer uma certa organização, facilita muito durante a mudança de vários itens de uma só vez (principalmente em casos de o site expandir e novos itens serem adicionados), além de podermos reutilizá-lo para mais arquivos HTML.
