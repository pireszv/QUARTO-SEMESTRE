A utilização do CSS INLINE facilita na vizualização do style juntamente ao campo, tornando mais simples para ajustes rápidos. Além disso, permite que fique tudo em um só arquivo, o que otimiza tempo. Entretanto, não é interessante para projetos grandes uma vez que pode demonstrar um baixo desempenho comparado ao Css externo e se tornar mais complexo se houverem muitos itens que terão a mesma estilização. Também deixa o HTML menos intuitivo.
Já com a utilização do CSS EXTERNO, apesar de parecer mais complicado para páginas muito pequenas/protótipos e requerer uma certa organização, facilita muito durante a mudança de vários itens de uma só vez (principalmente em casos do site expandir e novos itens serem adicionados), além de podermos o reutilizar para mais arquivos HTML.
